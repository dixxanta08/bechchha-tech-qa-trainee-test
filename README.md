# Bechchha Tech's QA Trainee Test

This is the assessment for the QA Trainee position at Bechchha Tech.

## How to Run the Application

1.  **Install Dependencies**:
    Open your terminal in the project directory and run:
    ```bash
    npm install
    ```

2.  **Start the Server**:
    Run the following command:
    ```bash
    npm start
    ```
    Or manually:
    ```bash
    node server.js
    ```

3.  **Access the App**:
    Open your web browser and visit:
    [http://localhost:3000/login.html](http://localhost:3000/login.html)

## Submission Instructions

You need to submit a **test document** containing the bugs you found.
*   **Format**: The document must include a **table** listing the bugs.
*   **Submission**: Send the document to the mail.

Good luck!
